## 1. 四旋翼动力学模型

### 1.1 坐标系与状态定义

**世界坐标系** `{W}`：NED（北-东-下）或 ENU（东-北-上）

**机体坐标系** `{B}`：固连于四旋翼质心

**状态向量**（12 维）：
```
x = [ξ, η]^T
ξ = [x, y, z, ẋ, ẏ, ż]^T    （位置与速度）
η = [φ, θ, ψ, p, q, r]^T    （姿态与角速率）
```

**控制输入**（4 维）：
```
u = [f, τ_φ, τ_θ, τ_ψ]^T
f = Σfᵢ                      （总推力）
τ_φ = l(f₂ - f₄)            （滚转力矩）
τ_θ = l(f₃ - f₁)            （俯仰力矩）
τ_ψ = c(f₁ - f₂ + f₃ - f₄) （偏航力矩）
```

### 1.2 平移动力学

```
mξ̈ = R·[0, 0, f]^T + [0, 0, -mg]^T + F_drag
```

展开为：
```
mẍ = f(sinφsinψ + cosφsinθcosψ) - k_drag·ẋ
mÿ = f(-sinφcosψ + cosφsinθsinψ) - k_drag·ẏ
mz̈ = f(cosφcosθ) - mg - k_drag·ż
```

### 1.3 旋转动力学

```
J·ω̇ = τ - ω × (J·ω)
```

展开为：
```
J_x·ṗ = τ_φ - (J_z - J_y)·q·r
J_y·q̇ = τ_θ - (J_x - J_z)·p·r
J_z·ṙ = τ_ψ - (J_y - J_x)·p·q
```

### 1.4 运动学方程

**欧拉角速率**（小角度近似）：
```
φ̇ ≈ p + q·sinφ·tanθ + r·cosφ·tanθ
θ̇ ≈ q·cosφ - r·sinφ
ψ̇ ≈ q·sinφ/cosθ + r·cosφ/cosθ
```

**四元数表示**（避免万向锁）：
```
q̇ = ½·Ω(ω)·q

Ω(ω) = [ 0   -p   -q   -r]
        [ p    0    r   -q]
        [ q   -r    0    p]
        [ r    q   -p    0]
```

### 1.5 小角度线性化（悬停点）

在悬停状态下（φ, θ, ψ ≈ 0，p, q, r ≈ 0），动力学可线性化为：

**姿态环**（解耦）：
```
φ̈ = τ_φ / J_x
θ̈ = τ_θ / J_y
ψ̈ = τ_ψ / J_z
```

**位置环**（Z 轴独立）：
```
z̈ = f/m - g ≈ (f_eq + Δf)/m - g
ẍ ≈ g·θ
ÿ ≈ -g·φ
```

## 2. PID 控制器推导

### 2.1 级联 PID 结构

```
外环（位置）：   x_ref → [PID_pos] → θ_ref, φ_ref
内环（姿态）：   θ_ref → [PID_att] → τ_θ
               φ_ref → [PID_att] → τ_φ
               ψ_ref → [PID_att] → τ_ψ
```

### 2.2 PID 控制律

**连续形式**：
```
u(t) = Kp·e(t) + Ki·∫e(τ)dτ + Kd·ė(t)
```

**离散形式（梯形积分）**：
```
u[k] = Kp·e[k] + Ki·Ts·Σ(e[i]+e[i-1])/2 + Kd·(e[k]-e[k-1])/Ts
```

### 2.3 传递函数

**开环传递函数**（以俯仰角为例）：
```
G_ol(s) = (Kp + Ki/s + Kd·s) · (1/(J_y·s²))
        = (Kd·s² + Kp·s + Ki) / (J_y·s³)
```

**闭环传递函数**：
```
G_cl(s) = G_ol/(1 + G_ol) = (Kd·s² + Kp·s + Ki) / (J_y·s³ + Kd·s² + Kp·s + Ki)
```

### 2.4 稳定性条件

由 Routh-Hurwitz 判据，特征方程 `J_y·s³ + Kd·s² + Kp·s + Ki = 0` 的稳定性条件：

```
Kd > 0, Kp > 0, Ki > 0
Kd·Kp > J_y·Ki
```

### 2.5 参数整定（Ziegler-Nichols）

**步骤**：
1. 设 Ki = 0, Kd = 0
2. 逐渐增大 Kp 直到系统等幅振荡
3. 记录临界增益 Ku 和振荡周期 Tu
4. 计算参数：

| 控制器类型 | Kp | Ki | Kd |
|-----------|-----|-----|-----|
| P | 0.5Ku | - | - |
| PI | 0.45Ku | 0.54Ku/Tu | - |
| PID | 0.6Ku | 1.2Ku/Tu | 0.075Ku·Tu |

## 3. ADRC 控制器推导

### 3.1 系统建模

将四旋翼姿态系统建模为二阶系统：
```
ÿ = f(y, ẏ, w, t) + b·u
```

其中：
- `f`：总扰动（包含内部动力学 + 外部风扰）
- `b`：对象增益（对于姿态环，b ≈ 1/J）
- `u`：控制输入（力矩）

### 3.2 ESO（扩展状态观测器）

**设计目标**：估计状态 `y, ẏ` 和总扰动 `f`

**ESO 方程**：
```
ė₁ = z₁ - y                          （观测误差）
ż₁ = z₂ - β₁·e₁                     （位置观测）
ż₂ = z₃ - β₂·e₁ + b₀·u             （速度观测 + 补偿）
ż₃ = -β₃·e₁                         （扰动观测）
```

**带宽参数化**：
```
β₁ = 3ω₀, β₂ = 3ω₀², β₃ = ω₀³
```

其中 `ω₀` 为观测器带宽（唯一调参）。

**离散形式（前向欧拉）**：
```
e₁[k] = z₁[k] - y[k]
z₁[k+1] = z₁[k] + Ts·(z₂[k] - β₁·e₁[k])
z₂[k+1] = z₂[k] + Ts·(z₃[k] - β₂·e₁[k] + b₀·u[k])
z₃[k+1] = z₃[k] + Ts·(-β₃·e₁[k])
```

### 3.3 收敛性分析

**观测误差动力学**：
令 `e_z1 = z₁ - y, e_z2 = z₂ - ẏ, e_z3 = z₃ - f`

```
ė_z1 = e_z2 - β₁·e_z1
ė_z2 = e_z3 - β₂·e_z1
ė_z3 = -ḣ - β₃·e_z1
```

**特征方程**：
```
s³ + β₁·s² + β₂·s + β₃ = (s + ω₀)³ = 0
```

**收敛条件**：
- 当 `ω₀ > 0` 时，所有特征值在左半平面
- 收敛速度由 `ω₀` 决定：越大越快，但噪声敏感
- 当 `ḣ = 0`（扰动变化率有限），观测误差趋于零

### 3.4 控制律设计

**NLSEF（非线性状态误差反馈）**：
```
e₁ = v₁ - z₁    （位置误差）
e₂ = v₂ - z₂    （速度误差）
u₀ = k·(e₁ + c·e₂)    （PD 形式）
```

**扰动补偿**：
```
u = (u₀ - z₃) / b₀
```

**闭环系统**（当 ESO 完美估计时）：
```
ÿ ≈ u₀ = k·(e₁ + c·ė₁)
```

等效为二阶系统：`G(s) = k·(1 + c·s) / (s² + k·c·s + k)`

取 `k = ωc², c = 2ζ/ωc`（ζ = 0.707 为阻尼比），则：
```
G(s) = ωc² / (s² + 2ζ·ωc·s + ωc²)
```

### 3.5 参数整定规则

| 参数 | 含义 | 整定规则 | 建议范围 |
|------|------|---------|---------|
| ω₀ | 观测器带宽 | 观测速度 | 10-100 rad/s |
| ωc | 控制器带宽 | 响应速度 | ω₀/3 ~ ω₀ |
| b₀ | 对象增益 | 粗略估计 | 1/J（允许 ±50%） |

## 4. MPC 控制器推导

### 4.1 系统离散化

将姿态环线性化后离散化（Ts = 0.02s）：
```
x[k+1] = A·x[k] + B·u[k]
y[k] = C·x[k]
```

以俯仰角为例：
```
x = [θ, θ̇]^T, u = τ_θ

A = [1  Ts]    B = [0     ]    C = [1  0]
    [0   1]        [Ts/J_y]
```

### 4.2 预测方程

**N 步预测**（N = 20）：
```
X = F·x[k] + Φ·U

其中：
X = [x[k+1], x[k+2], ..., x[k+N]]^T  （2N × 1）
U = [u[k], u[k+1], ..., u[k+N-1]]^T  （N × 1）

F = [A; A²; A³; ...; A^N]              （2N × 2）
Φ = [B    0    ...  0  ]
    [AB   B    ...  0  ]
    [A²B  AB   ...  0  ]
    [           ...    ]
    [A^(N-1)B  ...  B ]                （2N × N）
```

### 4.3 优化问题

**目标函数**：
```
J = Σᵢ₌₁ᴺ ‖x[k+i] - x_ref‖²_Q + Σᵢ₌₀^(N-1) ‖Δu[k+i]‖²_R
```

展开为标准 QP 形式：
```
min  ½·U^T·H·U + f^T·U

s.t. G·U ≤ w    （约束）
```

其中：
```
H = Φ^T·Q̃·Φ + R̃
f = Φ^T·Q̃·(F·x[k] - X_ref)
```

### 4.4 约束设计

| 约束类型 | 数学形式 | 物理意义 |
|---------|---------|---------|
| 姿态约束 | \|φ\|, \|θ\| ≤ 30° | 防止大角度失稳 |
| 执行器约束 | 0 ≤ f ≤ f_max | 电机推力限制 |
| 变化率约束 | \|Δu\| ≤ Δu_max | 执行器响应速率 |
| 状态约束 | \|z\| ≥ z_min | 最小飞行高度 |

### 4.5 QP 求解

**Active-Set 方法**：
1. 从无约束解 U* = -H⁻¹·f 开始
2. 检查约束违反情况
3. 添加最违反的约束到 active set
4. 在 active set 约束下重新求解
5. 重复直到无约束违反

**计算复杂度**：O(N³) per step（N = 20 时约 5-15ms）

## 5. PPO 算法推导

### 5.1 策略梯度基础

**目标函数**：
```
J(θ) = E_τ~π_θ[Σₜ γᵗ·rₜ] = E_τ~π_θ[R(τ)]
```

**策略梯度定理**：
```
∇J(θ) = E_τ~π_θ[Σₜ ∇log π_θ(aₜ|sₜ)·Âₜ]
```

其中 `Âₜ` 为优势函数估计。

### 5.2 TRPO 的问题

**TRPO 目标**：
```
max E[π_θ(a|s)/π_θ_old(a|s) · Â]
s.t. KL(π_θ_old || π_θ) ≤ δ
```

**问题**：KL 约束的优化困难，需要二阶优化（共轭梯度 + 线搜索）

### 5.3 PPO-Clip 推导

**概率比**：
```
r(θ) = π_θ(a|s) / π_θ_old(a|s)
```

**裁剪目标**：
```
L^CLIP(θ) = E[min(r(θ)Â, clip(r(θ), 1-ε, 1+ε)Â)]
```

**梯度分析**：

令 `L₁ = r(θ)Â, L₂ = clip(r(θ), 1-ε, 1+ε)Â`

**Case 1: Â > 0（好的动作）**
```
L₁ = r·Â （随 r 增大）
L₂ = (1+ε)·Â （有上界）

当 r < 1+ε 时：L = L₁，∇L = Â·∇r > 0（增大 π(a|s)）
当 r ≥ 1+ε 时：L = L₂，∇L = 0（停止更新）
```

**Case 2: Â < 0（差的动作）**
```
L₁ = r·Â （随 r 增大而减小）
L₂ = (1-ε)·Â （有下界）

当 r > 1-ε 时：L = L₁，∇L = Â·∇r < 0（减小 π(a|s)）
当 r ≤ 1-ε 时：L = L₂，∇L = 0（停止更新）
```

### 5.4 GAE（广义优势估计）

**TD 误差**：
```
δₜ = rₜ + γ·V(sₜ₊₁) - V(sₜ)
```

**GAE 公式**：
```
Â_t^GAE(γ,λ) = Σₗ₌₀^∞ (γλ)ˡ·δₜ₊ₗ
             = δₜ + (γλ)·δₜ₊₁ + (γλ)²·δₜ₊₂ + ...
```

**展开形式**：
```
Â_t = Σₗ₌₀^(T-t) (γλ)ˡ·(rₜ₊ₗ + γ·V(sₜ₊ₗ₊₁) - V(sₜ₊ₗ))
```

**偏差-方差权衡**：
- λ = 0：Â = δₜ（低方差，高偏差）
- λ = 1：Â = Σγˡrₜ₊ₗ - V(sₜ)（高方差，低偏差）
- λ = 0.95：折中选择

### 5.5 PPO 完整算法

```
初始化 θ, φ（策略和价值网络参数）
for iteration = 1, 2, ... do
    // 采样阶段
    用 π_θ_old 收集 N 条轨迹 {τ₁, ..., τₙ}
    for each (sₜ, aₜ, rₜ, sₜ₊₁) do
        计算 Â_t^GAE
        计算 R_t = Â_t + V_φ(sₜ)
    end

    // 更新阶段
    for epoch = 1, ..., K do
        for mini-batch B in {τ₁, ..., τₙ} do
            计算 r(θ) = π_θ(a|s) / π_θ_old(a|s)
            L^CLIP = E[min(r(θ)Â, clip(r(θ))Â)]
            L^VF = E[(V_φ(s) - R)²]
            H = E[-log π_θ(a|s)]

            L = L^CLIP - c₁·L^VF + c₂·H
            θ ← θ + α·∇L
        end
    end
    θ_old ← θ
end
```

## 6. SAC 算法推导

### 6.1 最大熵框架

**目标函数**：
```
J(π) = E_τ~π[Σₜ γᵗ·(rₜ + α·H(π(·|sₜ)))]
```

**软贝尔曼方程**：
```
Q^π(s,a) = r(s,a) + γ·E_{s'}[V^π(s')]
V^π(s) = E_{a~π}[Q^π(s,a) - α·log π(a|s)]
```

### 6.2 软 Q 函数更新

**目标 Q 值**：
```
y = r + γ·(Q_φ̄(s', ã') - α·log π_θ(ã'|s'))
ã' ~ π_θ(·|s')
```

**损失函数**：
```
L(φ) = E[(Q_φ(s,a) - y)²]
```

### 6.3 策略更新（重参数化技巧）

**采样**：
```
ε ~ N(0, I)
μ, σ = f_θ(s)
a = tanh(μ + σ ⊙ ε)    （重参数化）
```

**策略损失**：
```
L(θ) = E_{s~D, ε~N}[α·log π_θ(a|s) - Q_φ(s,a)]
     = E[α·log π_θ(a|s) - Q_φ(s, tanh(μ + σ⊙ε))]
```

**梯度**：
```
∇L(θ) = E[α·∇log π_θ(a|s) - ∇Q_φ(s,a)]
```

### 6.4 自动温度调节

**温度损失**：
```
L(α) = -α·E_{a~π}[log π(a|s) + H̄]
```

其中 `H̄ = -dim(A)` 为目标熵。

**最优 α 的解析解**：
```
α* = argmin L(α) → E[log π(a|s)] = -H̄
```

**梯度更新**：
```
α ← α - η_α·∇L(α)
  = α - η_α·(-E[log π(a|s) + H̄])
```

### 6.5 SAC 完整算法

```
初始化 θ, φ, φ̄, α（策略、Q网络、目标Q网络、温度）
初始化回放池 D
for step = 1, 2, ... do
    // 采样
    a_t ~ π_θ(·|s_t)
    执行 a_t，观测 r_t, s_{t+1}
    D ← D ∪ {(s_t, a_t, r_t, s_{t+1})}

    // 更新
    for each gradient step do
        从 D 采样 mini-batch B
        // Q 网络更新
        y = r + γ·(min(Q_φ̄₁, Q_φ̄₂)(s', ã') - α·log π_θ(ã'|s'))
        L(φᵢ) = E[(Q_φᵢ(s,a) - y)²], i=1,2
        φᵢ ← φᵢ - η_φ·∇L(φᵢ)

        // 策略更新
        L(θ) = E[α·log π_θ(ã|s) - min(Q_φ₁, Q_φ₂)(s, ã)]
        θ ← θ - η_θ·∇L(θ)

        // 温度更新
        L(α) = -α·E[log π_θ(ã|s) + H̄]
        α ← α - η_α·∇L(α)

        // 目标网络软更新
        φ̄ ← τ·φ + (1-τ)·φ̄
    end
end
```

## 7. 算法复杂度对比

### 7.1 计算复杂度

| 算法 | 每步计算 | 浮点运算量 | 典型延迟 |
|------|---------|-----------|---------|
| PID | 3 次乘法 + 2 次加法 | ~20 FLOPs | <0.01ms |
| ADRC | ESO (12 次乘法) + 控制律 (5 次) | ~50 FLOPs | <0.05ms |
| MPC | QP 求解 (N³/3) | ~2700 FLOPs (N=20) | 5-15ms |
| PPO | 神经网络前向 (64×64×4) | ~8K FLOPs | ~1ms |
| SAC | 神经网络前向 (256×256×4) | ~130K FLOPs | ~2ms |

### 7.2 参数数量

| 算法 | 参数 | 整定方式 |
|------|------|---------|
| PID | 3 × 3轴 = 9 | Ziegler-Nichols / 手工 |
| ADRC | 3 × 3轴 = 9 | 带宽参数化 |
| MPC | Q(12), R(4), N(1) ≈ 17 | 权重矩阵设计 |
| PPO | ~5K（网络） + 8（超参） | 网格搜索 |
| SAC | ~70K（网络） + 7（超参） | 自动温度 |

### 7.3 理论保证

| 算法 | 稳定性 | 鲁棒性 | 最优性 |
|------|--------|--------|--------|
| PID | Routh-Hurwitz | 增益裕度/相位裕度 | 局部最优 |
| ADRC | ESO 收敛 + 等效系统 | b₀ 容忍 ±50% | 近似最优 |
| MPC | 约束满足 | 显式约束处理 | 滚动最优 |
| PPO | 裁剪限制更新步长 | 无理论保证 | 近似最优 |
| SAC | 软贝尔曼收敛 | 最大熵探索 | 软最优 |

| 优先级 | 内容 | 原因 |
|--------|------|------|
| P0 | 四旋翼动力学模型 | 所有算法的共同基础 |
| P0 | PID 姿态控制 | 最简单的基线 |
| P1 | ADRC 姿态控制 | 验证抗扰优势 |
| P1 | MPC 姿态控制 | 验证约束处理 |
