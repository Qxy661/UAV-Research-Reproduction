# 第三章 方法论

## 3.1 四旋翼动力学模型

### 3.1.1 12 状态模型

四旋翼的完整状态向量为：

```
x = [x, y, z, u, v, w, φ, θ, ψ, p, q, r]
```

其中位置 (x,y,z)、速度 (u,v,w)、欧拉角 (φ,θ,ψ)、角速率 (p,q,r)。

平移动力学：
```
m·[u̇, v̇, ẇ] = R·[0, 0, f]ᵀ + [0, 0, -m·g]ᵀ - k_drag·[u, v, w]ᵀ
```

旋转动力学：
```
J·ω̇ = τ - ω × (J·ω)
```

### 3.1.2 6DOF 简化模型

用于 RL 训练的简化模型（8 维状态）：

```
状态: [z_err, vz, φ, θ, ψ, p, q, r]
动作: [thrust_cmd, τ_φ, τ_θ, τ_ψ] ∈ [-1, 1]
```

高度动力学：
```
az = (f·cos(φ)·cos(θ) - m·g + wind) / m
```

姿态动力学（简化的欧拉方程）：
```
dp = (τ_φ - (Jz-Jy)·q·r) / Jx
```

## 3.2 传统控制方法

### 3.2.1 PID 控制

级联 PID 结构：

- **外环**（姿态角 → 角速率参考）：Kp=6.0, Ki=0.1, Kd=0.3
- **内环**（角速率 → 力矩）：Kp=0.015, Ki=0.001, Kd=0.0001
- **高度环**：Kp=10.0, Ki=0.5, Kd=5.0

抗饱和：积分限幅 ±0.5

### 3.2.2 ADRC

ESO（扩张状态观测器）带宽参数化：

```
β₁ = 3ω₀, β₂ = 3ω₀², β₃ = ω₀³
```

参数：ω₀=50（ESO 带宽），ωc=15（控制器带宽）

### 3.2.3 MPC

单通道 MPC，预测步长 N=20：

```
Q = [10, 0; 0, 1], R = 0.1, dR = 0.01
```

使用 quadprog 求解 QP 问题。

## 3.3 强化学习方法

### 3.3.1 PPO

- Clip 机制：ε=0.2
- GAE：λ=0.95, γ=0.99
- 网络：128 hidden, lr=3e-4
- 熵系数：0.01

### 3.3.2 SAC

- 双 Q 网络 + 自动温度调节
- 网络：256 hidden, lr=3e-4
- 软更新：τ=0.005
- Replay buffer：100000

### 3.3.3 奖励设计

分层奖励函数：

| 条件 | 奖励 |
|------|------|
| z_err < 0.05m 且 att_err < 0.01 | +10（精确悬停） |
| z_err < 0.1m 且 att_err < 0.05 | +5（良好悬停） |
| z_err < 0.3m | 1.0 - z_err×5（接近） |
| z_err ≥ 0.3m | -z_err×5（远离） |

附加惩罚：速度、角速率、控制努力
