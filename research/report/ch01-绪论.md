# 第一章 绪论

## 1.1 研究背景

无人机（UAV）在军事侦察、农业植保、物流配送、应急救援等领域的广泛应用，对飞行控制系统的性能提出了越来越高的要求。传统的 PID 控制器因其结构简单、调参直观，长期占据工程实践的主导地位。然而，面对复杂飞行环境（强风扰动、参数不确定性、动态障碍物），PID 的固定增益特性限制了其适应能力。

近年来，两类新兴方法为无人机控制带来了新的可能性：

1. **自抗扰控制（ADRC）**：由韩京清研究员提出，通过扩张状态观测器（ESO）在线估计并补偿总扰动，不依赖精确模型即可实现高性能控制。
2. **模型预测控制（MPC）**：基于滚动优化和约束处理，在满足安全约束的前提下实现最优控制。
3. **深度强化学习（DRL）**：PPO、SAC 等算法通过与环境交互学习策略，有望在未知环境中实现自适应控制。
4. **大语言模型（LLM）**：GPT-4、Claude 等模型展现了强大的语义理解和推理能力，可用于高层任务规划和异常处理。

## 1.2 研究动机

尽管上述方法各有优势，但缺乏系统性的横向对比：

- **控制方法间**：PID、ADRC、MPC 在同一平台上的响应速度、抗扰能力、鲁棒性如何？
- **RL 方法间**：PPO（On-policy）与 SAC（Off-policy）在无人机悬停任务上的性能差异？
- **跨方向**：传统控制与 RL 在同一任务上的表现如何？各有什么优势？
- **LLM 边界**：大语言模型在无人机任务规划中的能力边界在哪里？

## 1.3 研究目标

本研究以 Crazyflie 2.1 微型四旋翼为平台，系统对比三类方法：

1. **控制方向**：PID、ADRC、MPC 的阶跃响应、风扰抑制、参数敏感度
2. **RL 方向**：PPO、SAC 在 6DOF 悬停任务上的训练性能和泛化能力
3. **跨方向**：在同一环境中统一评估传统控制与 RL 的优劣
4. **LLM 方向**：任务分解和异常处理的能力评估

## 1.4 主要贡献

1. 建立了包含文献综述、理论推导、代码复现、实验验证的完整研究框架
2. 在 MATLAB 和 Python 中分别实现了 PID/ADRC/MPC 和 PPO/SAC 的全栈复现
3. 设计了跨方向统一基准，首次在同一环境中对比传统控制与 RL
4. 提出了基于分层奖励的 6DOF 悬停环境，SAC 达到 0.019m 位置误差
5. 系统评估了 LLM 在无人机任务规划中的能力边界
